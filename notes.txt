mlp for action chunks: 

input: noised action chunk (B, length, dof) 
output: embedded action chunk (B, length, action_dim (usually 512-768))

mlp for proprio: 

input: proprio vector (B, dof)
output: proprio token (B, 1, action_dim)

action token: concat proprio token (B, 1, action_dim) + embedded action chunk (B, length, action_dim) = action token (B, length + 1, action_dim) 

positional embeddings: 

pos embed (max_seq_len, dim), add it element wise to the concatenated action token 

idm 

input: vae encoded video chunk: (length, embed_dim), action noise level: (tau), noised action chunk at time t: (length, dof)
output: epsilon: (length, embed_dim) 

loss function: MSE(action chunk at noise t - action chunk at noise t+1)
